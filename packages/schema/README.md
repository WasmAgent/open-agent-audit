# @openagentaudit/schema

TypeScript types and JSON schema artifacts for the OpenAgentAudit
canonical evidence model.

**Status:** alpha skeleton. The authoritative JSON schemas live at
`schemas/v0.1/*.schema.json` in the repository root. This package
re-exports TypeScript types and (when the freeze gate clears) a runtime
validator.

See [`docs/schema-versioning.md`](../../docs/schema-versioning.md) for
when the Phase 2 implementation is unblocked.
